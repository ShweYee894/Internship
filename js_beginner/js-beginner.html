Step 1 of 9

📦 Variables & Syntax

Variables are containers that store data values. JavaScript gives you three ways to declare them.

var — The Old Way

var is function-scoped and can be re-declared. It's the original keyword but is rarely used in modern code.

JavaScript 
var greeting = "Hello";
console.log(greeting); // "Hello"

var greeting = "Hi again"; // allowed with var
console.log(greeting); // "Hi again"

let — Reassignable Variable

let is block-scoped. Use it when the value will change later.

JavaScript 
let score = 0;
console.log(score); // 0

score = 10; // reassignment is OK
console.log(score); // 10

const — Constant Value

const is also block-scoped but cannot be reassigned. Use it for values that should not change.

JavaScript 
const PI = 3.14159;
console.log(PI); // 3.14159

const APP_NAME = "JSLearn";
console.log(APP_NAME); // "JSLearn"
💡 Best practice: Always use const by default. Switch to let only when you need to reassign. Avoid var.
Step 2 of 9

🏷️ Data Types

JavaScript has several primitive data types. Each stores a different kind of value.

The Five Primitives

Type Example Description
String "Hello" Text wrapped in quotes
Number 42, 3.14 Integers and decimals
Boolean true / false Logical true or false
null null Intentional empty value
undefined undefined Variable declared but not assigned

Checking Types with typeof

JavaScript 
console.log(typeof "Hello");     // "string"
console.log(typeof 42);          // "number"
console.log(typeof true);        // "boolean"
console.log(typeof null);        // "object" (known JS quirk!)
console.log(typeof undefined);   // "undefined"
⚠️ typeof null returns "object" — this is a historic bug in JavaScript that was never fixed!
Step 3 of 9

⚙️ Operators

Operators let you perform calculations, compare values, and combine conditions.

Arithmetic Operators

JavaScript 
let a = 10, b = 3;

console.log(a + b);  // 13  (addition)
console.log(a - b);  // 7   (subtraction)
console.log(a * b);  // 30  (multiplication)
console.log(a / b);  // 3.33 (division)
console.log(a % b);  // 1   (remainder)
console.log(a ** b); // 1000 (exponent)

Comparison Operators

JavaScript 
console.log(5 ==  "5");  // true  (loose — converts type)
console.log(5 === "5");  // false (strict — checks type too)
console.log(5 !=  "5");  // false
console.log(5 !== "5");  // true
console.log(10 > 3);    // true
console.log(10 <= 10);  // true

Logical Operators

JavaScript 
let x = true, y = false;

console.log(x && y);  // false (AND — both must be true)
console.log(x || y);  // true  (OR  — one must be true)
console.log(!x);      // false (NOT — flips the value)
💡 Always use === (strict equality) instead of == to avoid unexpected type-coercion bugs.
Step 4 of 9

🔀 Conditional Statements

Conditionals let your code make decisions based on different conditions.

if / else if / else

JavaScript 
let age = 18;

if (age >= 18) {
  console.log("You can vote! 🗳️");
} else if (age >= 16) {
  console.log("Almost there!");
} else {
  console.log("Too young to vote.");
}

switch Statement

Use switch when comparing one value against many possible cases.

JavaScript 
let day = "Monday";

switch (day) {
  case "Monday":
    console.log("Start of the week 💪");
    break;
  case "Friday":
    console.log("Almost weekend! 🎉");
    break;
  default:
    console.log("Just another day.");
}
Step 5 of 9

🔁 Loops

Loops let you repeat a block of code multiple times without writing it over and over.

for Loop

JavaScript 
for (let i = 1; i <= 5; i++) {
  console.log("Count: " + i);
}

while Loop

JavaScript 
let count = 3;

while (count > 0) {
  console.log("Countdown: " + count);
  count--;
}
console.log("Go! 🚀");

forEach — Loop Over Arrays

JavaScript 
const fruits = ["Apple", "Banana", "Cherry"];

fruits.forEach(function(fruit) {
  console.log("I like " + fruit);
});
Step 6 of 9

🔧 Functions

Functions are reusable blocks of code. JavaScript offers three ways to create them.

Function Declaration

The classic way — hoisted, meaning you can call it before it's defined in the code.

JavaScript 
function greet(name) {
  return "Hello, " + name + "!";
}

console.log(greet("Alice"));
console.log(greet("Bob"));

Function Expression

A function stored in a variable. Not hoisted — must be defined before use.

JavaScript 
const add = function(a, b) {
  return a + b;
};

console.log(add(5, 3));  // 8
console.log(add(10, 20)); // 30

Arrow Function =>

A shorter syntax introduced in ES6. Great for simple, one-line operations.

JavaScript 
const multiply = (a, b) => a * b;
console.log(multiply(4, 5)); // 20

const square = n => n * n;
console.log(square(7));      // 49

// Multi-line arrow function
const describe = (name, age) => {
  return name + " is " + age + " years old.";
};
console.log(describe("Sara", 25));
Step 7 of 9

📋 Arrays

Arrays store ordered lists of values. They come with many built-in methods.

Creating & Accessing Arrays

JavaScript 
const colors = ["Red", "Green", "Blue"];

console.log(colors[0]);      // "Red"   (index starts at 0)
console.log(colors[2]);      // "Blue"
console.log(colors.length); // 3

Common Array Methods

JavaScript 
const nums = [1, 2, 3];

nums.push(4);        // add to end
console.log(nums);   // [1, 2, 3, 4]

nums.pop();          // remove last
console.log(nums);   // [1, 2, 3]

console.log(nums.includes(2)); // true
console.log(nums.indexOf(3));  // 2

// map — transform each element
const doubled = nums.map(n => n * 2);
console.log(doubled); // [2, 4, 6]

// filter — keep elements that pass a test
const big = nums.filter(n => n > 1);
console.log(big);     // [2, 3]
Step 8 of 9

🧱 Objects

Objects store data as key–value pairs, perfect for representing real-world things.

Creating & Accessing Objects

JavaScript 
const user = {
  name: "Alice",
  age: 25,
  isStudent: true
};

// Dot notation
console.log(user.name);  // "Alice"

// Bracket notation
console.log(user["age"]); // 25

// Modify a property
user.age = 26;
console.log(user.age);    // 26

Object Methods & Looping

JavaScript 
const car = {
  brand: "Toyota",
  year: 2024,
  describe() {
    return this.brand + " (" + this.year + ")";
  }
};

console.log(car.describe());

// Loop over keys
for (let key in car) {
  if (typeof car[key] !== "function") {
    console.log(key + ": " + car[key]);
  }
}
Step 9 of 9

🌐 DOM Manipulation

The DOM (Document Object Model) lets JavaScript interact with the HTML page — selecting, changing, and creating elements.

Selecting Elements

JavaScript 
// By ID
const title = document.getElementById("main-title");

// By CSS selector (first match)
const btn = document.querySelector(".my-button");

// All matches (returns a NodeList)
const items = document.querySelectorAll("li");

Changing Content & Styles

JavaScript 
// Change text
title.textContent = "New Title!";

// Change HTML inside an element
title.innerHTML = "<em>Styled Title</em>";

// Change CSS
title.style.color = "#6c63ff";
title.style.fontSize = "2rem";

Event Listeners — Live Demo

Click the button below to see an event listener in action!

JavaScript 
// HTML: <button id="demo-btn">Click Me</button>
//        <p id="demo-output"></p>

const demoBtn = document.getElementById("demo-btn");
let clicks = 0;

demoBtn.addEventListener("click", () => {
  clicks++;
  document.getElementById("demo-output")
    .textContent = "Clicked " + clicks + " time(s)!";
});

Creating & Appending Elements

JavaScript 
// Create a new element
const newItem = document.createElement("p");
newItem.textContent = "I was added by JavaScript!";
newItem.style.color = "#00e88f";

// Append to the page
document.body.appendChild(newItem);
